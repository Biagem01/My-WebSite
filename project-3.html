INTRODUCTION

The project consists of implementing a C++ program that reads student data from a text file called 'Studenti.txt', to create groups of students, following specific criteria based on grade point average using data structures.

DESCRIPTION OF THE PROBLEM

The 'Studenti.txt' file contains the following fields for each student: student number, surname, name, city of residence and grade point average. The objective is to read this data from files and store them in dynamic structures to allow the group creation operation according to pre-established criteria.

IMPLEMENTATION OF CLASSES

The Students class was implemented to represent the students of the program. It contains information such as student number, surname, first name, city of residence, and grade point average.

CONCLUSION

In conclusion, through the implementation of the 'Students', 'List', and 'Stack' classes, together with the main function, I was able to satisfy the established objectives:

  • The Students class was created to represent each student, with methods to access their attributes and comparison operators to facilitate sorting operations.
  • The List and Stack classes were designed to handle a linked list and a stack of Student objects, respectively, providing methods for inserting, searching, and extracting students.
  • The main function was developed to coordinate the entire process, loading data from the text file, creating groups of students and printing the results.

The project allowed you to become familiar with the concepts of object-oriented programming, the management of dynamic data structures and the manipulation of text files in C++. Additionally, it provided the opportunity to apply advanced concepts such as comparison operators and operator overloading to improve the readability and flexibility of the code.

INTRODUCTION

The project in question is a platform game developed using Unity and the C# programming language. The game, titled Knight Warrior, is based on a series of classic platform game mechanics. The player's main objective is to guide the protagonist through a series of levels, collecting strawberries and facing challenges.

SPECIFIC OBJECTIVES

The specific objectives of the project include: Create a series of well-designed and balanced levels that offer a gradual progression of difficulty. Implement innovative and interesting game mechanics, including collecting power-ups, time management, and interacting with enemies and obstacles. Integrate customization elements and gameplay options, allowing users to adjust audio settings, controls and other preferences to suit their needs and preferences. Checkpoints in Levels: Implement checkpoints within levels to allow the player to resume from certain points, ensuring a smoother and less frustrating gaming experience. Scoreboard: Create a user scoreboard, which tracks each player's performance based on the number of strawberries collected during the game. Custom Animations and Sounds: Enrich the gaming experience with original animations and custom sounds to make interaction with the game more engaging and immersive.

GENERAL SETTINGS

Splash Screen: A personalized splash screen that introduces the game to the user, creating an impression of anticipation and anticipation. Game Icon: A representative icon for the game, which helps identify and differentiate it within the device or gaming platform.

ADVANCED IMPLEMENTATIONS

  • C# Programming Language: Using the C# programming language for game development, ensuring solid structure and object-oriented programming.
  • PlayerPrefs: Using PlayerPrefs to manage user rankings and scores.
  • Singleton: Implementazione di Singleton per GameManager e SoundManager, garantendo l'accesso univoco a tali componenti all'interno del gioco.
  • Singleton: Singleton implementation for GameManager and SoundManager, ensuring unique access to those components within the game.
  • Inheritance and Overriding: Using inheritance and overriding to create and manage specialized behaviors for traps, enemy projectiles, and other game entities.
  • Custom Animations and Sounds: Integration of original animations and custom sounds to enrich the audiovisual experience of the game.
